# АК-2  Лабораторна робота №3

### Ключі:
-h (--help)

-v (--version)

-l (--list)

-n (--number)

### Приклади:

```
$ ./main -h
Use commands
-v
-l
-n
-o
```

```
$ ./main --number 5
Number you indicated is 5
```
```
$ ./main -vn5
Version: 1.0.0
Number you indicated is 5
```

```
$ ./main -v --version -hv -n7 --number 777
Version: 1.0.0
Use commands
-v
-l
-n
-o
Number you indicated is 7

```
